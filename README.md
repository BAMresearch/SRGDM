# SRGDM
Super-Resolution for Gas Distribution Mapping

(under construction)

This repository contains code for super-resolution gas distribution mapping.
More information can be found in: 
N. P. Winkler, H. Matsukura, P. P. Neumann, E. Schaffernicht, H. Ishida and A. J. Lilienthal, "Super-Resolution for Gas Distribution Mapping: Convolutional Encoder-Decoder Network," 2022 IEEE International Symposium on Olfaction and Electronic Nose (ISOEN), Aveiro, Portugal, 2022, pp. 1-3, doi: 10.1109/ISOEN54820.2022.9789555. [[1]](https://ieeexplore.ieee.org/abstract/document/9789555)

Winkler, N.P. et al. (2023). Learning from the Past: Sequential Deep Learning for Gas Distribution Mapping. In: Tardioli, D., Matellán, V., Heredia, G., Silva, M.F., Marques, L. (eds) ROBOT2022: Fifth Iberian Robotics Conference. ROBOT 2022. Lecture Notes in Networks and Systems, vol 590. Springer, Cham. https://doi.org/10.1007/978-3-031-21062-4_15
